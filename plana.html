Figuras Planas


Vamos estudar como calcular a área de algumas figuras planas importantes.

Triângulo Qualquer:



1° Modo: Quando se sabe a medida da base e da altura.
A = (Base . Altura) /2 = (a.h)/2

2° Modo: Quando se sabe a medida de todos os lados (fórmula de Heron).
p = Perímetro/2 = (a + b + c) /2 (semiperímetro)
A = √p(p - a).(p - b).(p - c)

Triângulo Equilátero:



A = (L2.√3)/4

Quadrado:



A = L2
Perímetro = 4.L

Retângulo:



A = b.h
Perímetro = 2.(a + b)

Losango:



Paralelogramo:


A = a.b

Trapézio:



A = (B + b).h/2

Círculo



Área da circunferência: A = π.r2
Comprimento da circunferência: C = 2.π.r
